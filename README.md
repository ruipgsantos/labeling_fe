# Labelling Front-end

To run, simply

- edit .env to your liking. If you do:
    - match REACT_APP_SERVER_URL to the server port
    - match PORT to the server CLIENT_URL, for CORS purposes
- run docker-compose up
- browse localhost:${PORT}
